# Customer Churn Prediction using Logistic Regression

## 📌 Project Overview
This project focuses on predicting whether a customer will churn (leave a service) using a Logistic Regression machine learning model. The goal is to identify customers at risk of leaving so that retention strategies can be applied.

---

## 📊 Problem Statement
Customer churn is a major problem for subscription-based businesses. This model helps predict churn based on customer behavior and usage patterns.

---

## 📁 Dataset
The dataset contains customer-related features such as:
- Tenure
- Monthly charges
- Total charges
- Contract type
- Customer demographics

---

## ⚙️ Methodology

### 1. Data Preprocessing
- Handling missing values
- Encoding categorical variables
- Feature scaling

### 2. Model Building
- Logistic Regression algorithm
- Train-test split applied

### 3. Evaluation Metrics
- Accuracy score
- Confusion matrix
- Precision and recall

---

## 📈 Results
- Model successfully predicts customer churn with reasonable accuracy
- Helps identify high-risk customers
